花密CLI版
=========

概述
----
作者：Tao Yang (<http://ninehills.github.com>)

原始项目：徐小花 kisexu(at)gmail.com <https://code.google.com/p/flower-password/>

本项目和huami官方的名词对应：

* Key -> 网站代号
* Password -> 记忆密码

项目依赖
--------
* Python 2.x

使用方法
--------

    $ ./huami.py KEY
    Password:
    KEF2342511421312

可以结合xclip，比如

    ./huami.py KEY | xclip -i

便可将密码复制到X剪切板。
